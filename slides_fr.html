Exemple d'instructions conditionnelles

Dans l'exemple ci-dessous, quelle valeur pourrait-on ajouter pour faire apparaître les deux messages? Qu'arrive-t-il si la valeur est de 100? Cliquez sur « Run » (Exécuter) et vérifiez votre hypothèse.

var prix = prompt("Combien coûte le robot?");
      
if ( prix <= 100 ) {
  alert(prix + "$" + " n'est pas cher. Je l'achète!");
} else {
  alert(prix + "$" + "? Mais c'est trop cher!");
}

<script src="">

Plus vous écrivez de blocs de JavaScript ou si vous travaillez avec une équipe, plus vous voudrez mettre les blocs de code JavaScript dans leurs propres fichiers pour les gérer plus facilement. Un site volumineux peut contenir des milliers de lignes de code écrites par plusieurs personnes!

Pour faire un lien vers un fichier de JavaScript externe, faites ceci :

<html>
  <head>
    <title>Exemple de JavaScript!
  </head>
  <body>
    <div id="example">Ceci est un exemple.
    <script type="text/javascript" src="scripts.js"></script>
  </body>
</html>
Le fichier externe doit être enregistré avec l'extension de fichier « .js ». Il n'est pas nécessaire d'ajouter les balises « <script> » à l'intérieur du fichier.
